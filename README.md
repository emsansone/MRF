Sync - C++ project (Visual Studio 2012)
Copyright, Emanuele Sansone 2016


The code implements the MRF stage described in the following paper:


"Automatic Synchronization of Multi-User Photo Galleries" 
to appear in IEEE Trans. on Multimedia (2017)


In order to run it:
1. Compile using Visual Studio (version >= 2012)
2. Add in the same folder of the final executable the files 
   "Parameters.txt" and "exiftool.exe"
